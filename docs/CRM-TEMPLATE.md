
# Leadflow Studio CRM-template

Gebruik dit als simpele leadtracker in Google Sheets, Notion, Airtable of een markdownbestand.

## Kolommen

| Kolom | Betekenis | Voorbeeld |
|---|---|---|
| Datum | Wanneer de lead binnenkwam | 2026-06-25 |
| Naam | Contactpersoon | Samira |
| Bedrijf | Bedrijfsnaam | Studio Snip |
| Niche | Type bedrijf | Kapper / salon |
| Website/Instagram | Link die je moet bekijken | https://... |
| Bron | Waar kwam de lead vandaan | Website, DM, referral |
| Aanbod | Audit, Starter Website, Website + Leadflow | Starter Website |
| Status | Huidige fase | Nieuw |
| Volgende actie | Wat moet jij doen | Mini-audit sturen |
| Deadline | Wanneer opvolgen | Morgen 10:00 |
| Waarde | Geschatte dealwaarde | €750 |
| Notities | Korte context | Wil meer afspraken via mobiel |

## Statussen

```text
Nieuw
Audit gestuurd
Follow-up nodig
Gesprek gepland
Voorstel gestuurd
Gewonnen
Verloren
Later opnieuw benaderen
```

## Simpele opvolgflow

1. Lead komt binnen via formulier.
2. Binnen 24 uur mini-audit of korte reactie sturen.
3. Na 2 dagen follow-up als er geen antwoord is.
4. Bij interesse Starter Website voorstel sturen.
5. Na voorstel altijd een concrete vervolgstap plannen.

## Follow-up tekst

```text
Hoi {{naam}}, korte check: heb je de mini-audit kunnen bekijken?

Als je wilt, kan ik de belangrijkste 2 punten ook omzetten naar een concreet Starter Website voorstel. Dan zie je meteen wat er nodig is, wat het kost en hoe snel het live kan.
```
