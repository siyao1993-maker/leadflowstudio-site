Security

Beveiliging en responsible disclosure.

Leadflow Studio is een statische website op Cloudflare. We nemen beveiliging serieus en willen kwetsbaarheden zorgvuldig oplossen.

Kwetsbaarheid melden

Zie je een beveiligingsprobleem? Mail dan naar:

Gebruik het beveiligde contactformulier

Vermeld bij voorkeur:

  • de URL of pagina waar je het probleem ziet;
  • een korte uitleg van het risico;
  • stappen om het probleem veilig te reproduceren;
  • geen destructieve acties of toegang tot gegevens van anderen.

Wat wij beschermen

  • HTTPS-only verkeer via Cloudflare;
  • security headers zoals CSP, HSTS, frame protection en permissions policy;
  • GitHub quality checks voordat wijzigingen worden gedeployed;
  • beperkte deploytoegang via repo-specifieke SSH key.

Over kopiëren van de website

Een publieke website kan technisch nooit volledig tegen kopiëren worden beschermd: bezoekers kunnen HTML, CSS en zichtbare teksten bekijken. Wel beschermen we het merk, de domeinnaam, de GitHub-toegang en de deployment pipeline zodat niemand zomaar de originele website kan aanpassen of kapotmaken.
